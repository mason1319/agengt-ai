# StarMate v4.1 内部自用版 | 未闭环控件清单

更新时间：2026-06-13

> 范围：继续上一轮首页 / 课程页 / 练习页 / 个人中心控件扫描，本轮重点补充课程表、课时、缴费记录展示口径，并按角色归档未闭环控件。
> 口径：Phase 1 只服务本机构内部运营，不做在线支付、公开购买、对外 SaaS 套餐售卖；用户侧统一使用“缴费记录 / 收费标准 / 入账课时”。

## 1. 本轮文案口径已收敛

| 页面 | 原口径 | 新口径 | 说明 |
|---|---|---|---|
| 家长中心 | 购买课时 | 入账课时 | 避免看起来像在线购买流程 |
| 家长中心 | 缴费摘要 / 缴费记录 | 缴费记录摘要 / 缴费记录 | 对齐内部收费留痕 |
| 课程页 | 费用 | 收费标准 | 对齐课程表展示，不暗示线上支付 |
| 个人中心 | 缴费状态 | 缴费记录 | 对齐课时账户与收费对账 |
| 公开课程 | 课程费用 | 收费标准 | 只展示咨询前标准，不做支付动作 |
| 全局空态 | 价格未设置 / 缴费记录未录入 | 收费标准未设置 / 缴费记录未录入 | 统一兜底词 |

## 7. 已确认的阶段口径

1. 家长端保留“缴费记录”模块，继续只读展示并保留追问入口。
2. 课程页当前只做课程表查看与核对，不新增独立创建页面。
3. 练习页 AI 生成内容统一落 `student_tasks`，用于复盘与回看。
4. 家校沟通稿入库使用 `parent_messages`，避免和课堂反馈混写。
