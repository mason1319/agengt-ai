# Aggie速记英语网页原型交付说明（给老板/客户）

## 1. 本次交付范围（只做网页版）

本次仅交付 Web 版原型，满足第一阶段目标：
- 5 个角色可切换：创始人 / 老师 / 家长 / 学生 / 平台运营
- 无需真实登录，使用模拟数据演示
- AI 为模拟输出（先可用）
- 面向移动端与桌面端响应式展示
- 当前不启动 iOS/Android/小程序正式壳体、不启动数据库建表

## 2. 你现在可直接看到的结果

- 首页有完整产品感页面，不是普通宣传页
- 创始人驾驶舱：今日必须处理、续费预警、在读学员、本月收入、今日消课、老师反馈未提交
- 老师工作台：今日课程、授权学生、一键消课、课堂记录、AI 反馈生成、AI 练习生成
- 家长端：成长卡、课程周报、进步项、待加强项、剩余课时、阶段报告
- 学生端：今日任务、单词/语法/阅读/错题/徽章
- 平台端：机构列表、试用到期、套餐状态、开通/延期/停用、AI 用量、学生/老师上限

## 3. 验收标准（本次先看）

- 页面可访问，不空白
- 角色切换正常（至少 5 个角色均可切到对应视图）
- 手机端不出现横向滚动条
- 老师端脱敏信息可见（非完整家长联系方式）
- 平台端可看到月费/年费/试用/到期与 AI 用量关系

## 4. 交付命令

```bash
cd /Users/mason/英语系统
npm install
npm run web:dev
npm run web:ship
```

`web:ship` 会输出交付目录：
- `delivery/web/StarMate-English-YYYYMMDD_HHMMSS/`
- 并同步 `delivery/web/latest` 指向最新版本
