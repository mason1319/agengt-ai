# StarMate English 网页验收截图清单（老板演示版）

建议顺序：先 PC 后手机，先基础页面再交互行为，便于一次性拍完。

## 1) PC 端（1920 或 1366）

1. 浏览器地址栏
   - 访问 `http://localhost:5173`
   - 确认页面不空白、顶部能看到左侧导航和角色切换

2. 创始人驾驶舱
   - 点击角色切换「创始人」
   - 截图：今日必须处理、续费预警、本月收入、在读学员、今日消课
   - 截图：新线索漏斗

3. 老师工作台
   - 点击角色切换「老师」
   - 截图：课程卡片列表 + 选中课程详情闭环区
   - 截图：家长信息脱敏区域（展示“手机号脱敏”）
   - 截图：一键消课 / AI反馈 / AI练习按钮状态

4. 家长端
   - 点击角色切换「家长」
   - 截图：成长卡+阶段完成度环
   - 截图：进步与加强建议、阶段报告导出按钮

5. 学生端
   - 点击角色切换「学生」
   - 截图：今日任务地图（2~3个任务节点）
   - 截图：积分徽章区

6. 平台运营后台
   - 点击角色切换「平台运营」
   - 截图：机构列表、试用/到期状态、套餐与AI用量
   - 截图：可执行动作（转正式、续费恢复、延长试用）

7. 页面附加区
   - 截图：AI 智能体介绍区（可选）
   - 截图：套餐页（体验版/基础版/标准版/专业版）

## 2) 手机端（375/390）

1. 主页适配
   - 开发者工具或真机访问 `http://192.168.x.x:5173`
   - 截图：无横向滚动条

2. 角色切换可用性
   - 截图：顶部角色切换栏完整显示（至少5个角色都能点）
   - 截图：导航栏横向滑动可见（如溢出时也不遮挡）

3. 关键交互
   - 选中老师课堂并点击“一键消课”后截图（状态变化）
   - 学生端点击“今日任务”后截图（完成率变化）
   - 平台端点击机构动作后截图（状态变化）

## 3) 交付命名建议

- 文件名示例：`star-mate-web-pc-founder-dashboard.png`
- 建议包含时间戳：`20260604-starmate-web-mobile-student.png`
- 统一放入一个目录供一次性提交给你审阅。
