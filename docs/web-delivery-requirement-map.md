## 验收建议

1. 先跑 `docs/web-delivery-checklist.md` 前 3 条（启动、显示、切换）
2. 再跑 `docs/web-delivery-screenshot-guide.md` 的手机和 PC 快照项
3. 最后输出 `npm run web:ship` 的交付目录给客户确认
