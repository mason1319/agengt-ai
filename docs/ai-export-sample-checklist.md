# AI 导出文件验收示例（样例）

> 用于快速抽检：`starmate-ai-usage-*.csv` 与 `starmate-ai-audit-*.csv`

## A. 机构用量导出样例（截取）

```csv
机构名称,套餐,套餐编码,窗口内AI消耗,月度配额,请求数,最后请求时间,AI来源,来源分类
测试小学A,标准版,standard,120,10000,240,2026-06-05 10:20,真实,real
测试小学B,基础版,basic,80,2000,120,2026-06-05 11:10,模拟,mock

[来源告警汇总],,,
指标,值,阈值,状态
整体来源风险,告警,模拟>=35% 或 未知>=15%,需要处理
真实来源请求占比,38%,-, -
模拟来源请求占比,45%,35%,告警
来源未知请求占比,17%,15%,告警
来源总请求,360,,-
真实来源请求,137,,-
模拟来源请求,162,,-
来源未知请求,61,,-
```

- 计算核对：
  - 模拟占比 = 162/360≈45%
  - 未知占比 = 61/360≈17%
  - 真实占比 = 137/360≈38%
  - 触发规则：45% >= 35% 或 17% >= 15% => `告警`

## B. 审计导出样例（截取）

```csv
时间,机构,机构ID,角色,用户ID,动作,决策,延迟(ms),Tokens,理由,来源,来源分类,IP,请求参数,响应参数
2026-06-05 10:00,测试小学A,org_001,teacher,t_001,ask,allow,120,36,正常处理,mock,mock,183.1.1.1,{"q":"..."},{"r":"..."}
2026-06-05 10:01,测试小学A,org_001,parent,p_001,ask,allow,98,40,降级返回,unknown,unknown,183.1.1.2,{"q":"..."},{"r":"..."}

[来源告警汇总],,,
指标,值,阈值,状态
整体来源风险,正常,模拟>=35% 或 未知>=15%,待观察
真实来源请求占比,70%,-, -
模拟来源请求占比,20%,35%,正常
来源未知请求占比,10%,15%,正常
来源总请求,10,,-
真实来源请求,7,,-
模拟来源请求,2,,-
来源未知请求,1,,-
```

- 计算核对：
  - 审计明细每条按 1 次请求计数
  - 实际比例：20%、10%、70%，未达阈值 => `正常`

## C. 简单验收清单
- 1) 文件有两个区块：明细 + `[来源告警汇总]`
- 2) 有 `整体来源风险` 一行
- 3) 模拟来源阈值列存在且与显示配置一致（默认35%）
- 4) 未知来源阈值列存在且与显示配置一致（默认15%）
- 5) `来源总请求` 与三类请求计数和一致
- 6) 状态与阈值计算结果一致
