# 星伴英语 StarMate 资质与备案清单 v4.1（中国境内）

**文档类型**：资质/备案/上线材料清单  
**状态**：内部草案，正式上线前需律师/备案服务商复核  
**日期**：2026-06-06  

---

## 1. 当前阶段结论

当前为本机构内部自用版，不做公开注册、支付、对外售卖和公开推广。

但只要通过公网域名向学生/家长提供服务，仍需评估 ICP 备案、公安联网备案、隐私政策展示、用户协议展示、未成年人授权等要求。

---

## 2. 资质/备案清单

| 项 | 当前是否需要 | 说明 |
|---|---|---|
| 营业执照/机构主体资料 | 需要 | 用于备案、协议主体、供应商合同 |
| 域名实名 | 需要 | 域名主体应与机构/负责人匹配 |
| ICP 备案 | 公网中国境内访问通常需要 | 视服务器/接入和域名使用情况办理 |
| 公安联网备案 | 上线后通常需要评估 | ICP 后按当地要求办理 |
| 增值电信业务许可证 | 当前通常不需要 | 若后期对外经营收费平台需法务判断 |
| 教育培训资质 | 需结合机构实际 | 若涉及线下/线上培训经营，应确认当地要求 |
| App 备案 | 当前不需要 | 当前不做 App |
| 小程序备案/平台审核 | 当前不需要 | 当前不做正式小程序 |
| 生成式 AI 服务备案/登记 | 当前需评估 | 如对公众提供生成式 AI 服务，需进一步评估 |
| 算法备案 | 当前需评估 | 如具有舆论属性或社会动员能力通常风险更高 |

## 3. 上线前材料

| 材料 | 用途 |
|---|---|
| 营业执照 | 主体证明 |

| 服务器/云服务信息 | 备案/安全评估 |

| 隐私政策 | 页面展示 |
| 家长授权书 | 未成年人使用 |
| 数据安全制度 | 内部管理 |
| 版权授权证明 | 教材/图片/音视频 |
| 第三方 AI/语音供应商协议 | 数据处理和责任边界 |

---

## 4. 公网页面底部建议

如果未来公网开放，应在页面底部展示：

1. 主体名称。
2. ICP 备案号。
3. 公安备案号。
4. 用户协议链接。
5. 隐私政策链接。
6. 未成年人保护/家长授权入口。
7. 投诉/联系邮箱或电话。

## 5. 后期商业化前置条件

未来如恢复商业版，应先补：

| 模块 | 文件/资质 |
|---|---|
| 公开注册 | 用户协议、隐私政策、账号注销机制 |
| 在线支付 | 支付协议、退款规则、发票规则 |
| SaaS 套餐 | 服务等级协议、订单合同、数据处理协议 |
| 多机构 | 机构入驻协议、管理员权限协议 |
| App | App 备案、应用商店材料、隐私合规检测 |
| 小程序 | 小程序平台审核材料 |
