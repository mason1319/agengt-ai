# 星伴英语 StarMate 版权与知识产权政策 v4.1（草案）

**适用范围**：课程内容、教材、图片、视频、音频、AI 输出、系统文档和代码
**状态**：内部草案，正式使用前需律师复核
**日期**：2026-06-06

---

## 1. 基本原则

1. 不使用未授权教材全文作为系统内容。
2. 不将第三方版权内容未经许可上传、改编、传播或商用。
3. 机构自建内容应保留来源和作者记录。
4. AI 生成内容上线前应经过抽检，避免侵权和不适龄内容。
5. 学生、老师、家长上传内容不得侵犯第三方权利。

---

## 2. 内容来源分类

| 来源 | 是否可用 | 条件 |
|---|---|---|
| 机构自建内容 | 可用 | 记录作者、日期、版本 |
| 已授权教材内容 | 可用 | 保存授权合同/证明 |
| 公开版权内容 | 谨慎可用 | 核实许可范围 |
| 第三方图片/视频/音频 | 谨慎可用 | 需授权或使用合规素材库 |
| 牛津/新概念等教材 | 不得直接复制全文 | 需授权，或仅做机构内部教学引用且保留合规评估 |
| 学生上传内容 | 可内部使用 | 不得对外公开 |
| AI 生成内容 | 可辅助使用 | 需审核，不保证无侵权风险 |

---

## 3. 教材使用规则

对国际音标、自然拼读、K12 同步、新概念、牛津等内容：

| 内容 | 规则 |
|---|---|
| 知识点 | 可自建讲解 |
| 题目 | 优先自编 |
| 课文全文 | 未授权不得直接复制入库 |
| 图片/音频 | 未授权不得上传 |
| 单元标签 | 可做内部课程映射 |
| 机构课堂材料 | 确认机构拥有使用权 |

---

## 4. AI 输出版权规则

1. AI 输出仅作为学习辅助内容。
2. 不直接声称 AI 输出绝对原创。
3. 重要材料上线前应人工抽检。
4. 不用 AI 大量仿写特定版权教材全文。
5. 不生成带有明确第三方版权风格、人物、品牌标识的内容用于公开展示。

---

## 5. 上传素材规则

| 素材 | 要求 |
|---|---|
| 老师公开视频 | 老师本人同意 |
| 学生照片/视频 | 家长同意 |
| 家长反馈 | 默认匿名或脱敏 |
| 机构活动照片 | 确认肖像和授权 |
| 第三方图片 | 需版权来源 |

第一阶段不正式上线完整文化墙；如后期开放，必须增加素材审核流程。

---

## 6. 权利归属

| 类型 | 建议归属 |
|---|---|
| 系统代码 | 按开发合同约定 |
| 产品文档 | 本机构/委托方所有或按合同约定 |
| 机构自建课程 | 本机构所有 |
| 老师原创内容 | 按劳动/合作关系约定 |
| 学生学习数据 | 依法受个人信息保护，不作为机构任意资产 |
| AI 生成反馈 | 作为教学辅助结果管理，使用权归机构内部教学场景 |

---

## 7. 侵权处理

发现疑似侵权时：

```text
下线内容
  → 记录来源
  → 核实授权
  → 联系权利人或删除
  → 复盘内容流程
```

严重侵权风险内容不得继续传播。
