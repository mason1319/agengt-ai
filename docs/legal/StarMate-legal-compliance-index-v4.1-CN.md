## 1. 合规定位

星伴英语当前不是对外商用 SaaS，不开放公开注册、支付、套餐购买、平台招商。当前仅用于本机构内部教学管理和 AI 英语学习。

但系统涉及未成年人、语音录音、学习数据、AI 生成反馈、家长授权、教材内容和文件存储，仍必须按中国境内合规标准设计。

v4.1 允许新增“招生咨询与 AI 客服接待”能力。该能力仅用于收集咨询线索、回答课程基础问题、预约试听和转人工跟进，不等同于公开注册、在线售卖或自动开通正式学习账号。

## 4. 当前必须执行的合规底线

| 底线 | 要求 |
|---|---|
| 家长授权 | 4-16 岁学生使用前必须由家长/监护人确认授权 |
| 语音采集说明 | 首次录音前明确说明用途、保存周期、删除方式 |
| 数据最小化 | 只采集教学必需数据 |
| 权限隔离 | 老师只能看授权学生，家长只能看自己孩子 |
| 内容安全 | AI 输出不得含违法、不良、成人、暴力、自残、歧视等内容 |
| 版权合规 | 未授权教材内容不得直接复制、传播、商用 |
| 备案预留 | 面向公网提供服务前，应完成 ICP/公安联网备案等必要手续 |
| 云服务评估 | 使用 Cloudflare 等境外/跨境服务前，需评估数据出境和访问路径 |
| 咨询合规 | 外部咨询必须有隐私告知、联系方式授权、反骚扰和退订/拒绝继续联系机制 |
| AI 客服边界 | 不承诺提分、不自动收费、不自动创建正式学生账号、不替代老师最终诊断 |
| 课程价格展示 | 可公开展示课程单价、连报价、课次、限招人数，但不得虚假宣传或隐藏重要收费条件 |
| 个人缴费隐私 | 学生个人缴费状态、实收金额、欠费情况仅本人家长和内部授权人员可见 |

## 5. 不可上线条件

出现以下任一情况，不允许正式给学生/家长使用：

| 条件 | 风险 |
|---|---|
| 无隐私政策 | 个人信息处理不透明 |
| 无家长授权 | 未成年人数据处理风险 |
| 录音无用途说明 | 敏感个人信息采集风险 |
| R2 文件裸公开 | 语音和图片泄露风险 |
| 老师可见未授权学生 | 权限越权 |
| 家长可见其他孩子 | 严重隐私风险 |
| 教材内容无授权 | 版权风险 |
| AI 输出无过滤 | 未成年人内容安全风险 |
| 咨询表单无隐私勾选 | 个人信息收集风险 |
| AI 客服承诺学习效果 | 虚假宣传和纠纷风险 |
| 公开其他学生缴费状态 | 隐私和名誉风险 |
